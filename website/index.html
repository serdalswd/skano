Features

Built for pixel artists
and retro lovers.

Every tool you need to create stunning dithered art, packed into a clean and fast app.

⚡

Live Preview

See changes instantly as you adjust settings. Background threading keeps the UI smooth at all times.

🎨

Custom Palettes

9 iconic built-in palettes — Game Boy, CGA, PICO-8, NES, Sweetie 16. Or build your own color set.

🔢

13 Algorithms

Floyd-Steinberg, Atkinson, Bayer matrices, Sierra, Stucki and more. Each with a unique character.

🎞

GIF Support

Process every frame of an animated GIF and export a perfectly dithered animation. Frame by frame.

🖼

Pixel Scale

Scale from 1× to 16×. Control the pixel size to get that chunky retro 8-bit or 16-bit look.

✂️

Fine Adjustments

Contrast, brightness, midtones and blur controls let you perfect the source before dithering.

13 dithering algorithms.

Error diffusion, ordered Bayer patterns, random noise — hover to highlight.
